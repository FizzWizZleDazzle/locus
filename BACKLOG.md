# Backlog

Ideas and future features, organized by category.

## Learning

- **Spaced repetition** — resurface problems the user previously got wrong, weighted by time since last attempt
- **Show solution on wrong ranked answers** — reveal the solution/editorial after an incorrect ranked submission
- **Topic prerequisite tree** — suggest topics to learn before tackling harder ones (e.g. algebra1 before algebra2)
- **Hint system for ranked/practice** — progressive hints that reduce ELO gain
- **Mistake analysis dashboard** — track common error patterns (sign errors, arithmetic, conceptual)
- **Custom problem sets/worksheets** — let users create and share curated problem collections

## Competition

- **Timed challenge mode (AMC format)** — 25 problems in 75 minutes, scoring like AMC (6 correct, -1.5 wrong, 0 blank)
- **Head-to-head duels** — real-time 1v1 with same problems, fastest correct answer wins

## Social

- **Friends/following** — follow other users, see their activity in a feed
- **Expanded achievements** — more badge categories, seasonal badges, community challenges

## Quality of Life

- **Keyboard shortcuts** — hotkeys for submit, next problem, toggle calculator
- **Mobile math input** — better touch-friendly math input for phones/tablets
- **Problem reporting/flagging** — let users flag broken, duplicate, or mis-graded problems

## Infrastructure

- **Difficulty calibration feedback loop** — adjust problem ELO based on aggregate user performance
- **Near-duplicate detection** — detect and merge problems that are structurally identical
- **Topic warmup mode** — review easier problems before jumping into ranked at current ELO
